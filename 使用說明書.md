# TypeTool-STT 語音轉文字 — 使用說明書

本機語音辨識工具,全程離線在地運算,輸出**繁體中文(台灣)**。
安裝於你 clone / 解壓縮的本資料夾內,四個功能全部雙擊或拖曳即可用。

---

## 快速上手

| 我要… | 做這個 |
|---|---|
| 用講的打字 | 雙擊 `語音輸入.bat`,按住 **Alt+Ctrl** 講、放開 |
| 錄會議轉逐字稿 | 雙擊 `錄會議.bat`,開會 → 按 Enter 停 |
| 手上已有音檔要轉文字 | 把音檔**拖到** `轉逐字稿_拖曳音檔到此.bat` |
| 轉文字並標記誰在講 | 把音檔**拖到** `說話人分離_拖曳音檔到此.bat` |

---

## 開頭都會問:用哪個模型?

每個工具啟動時跳選單,輸入 `1` 或 `2`(直接按 Enter＝1):

| | 1) SenseVoice | 2) Breeze-ASR-25 |
|---|---|---|
| 速度 | 極快(即時) | 較慢(每句多等約 0.5–1 秒) |
| 繁中 | OpenCC 轉台灣正體 | **原生台灣繁中** |
| 強項 | 通用、低延遲 | **台灣口語、中英夾雜** |
| 說話人分離 | ✅ 支援 | ❌ 不支援(改時間戳分行) |
| VRAM | ~1GB | ~3.3GB |

- 一般會議、要分辨誰在講 → **選 1**
- 中英夾雜多、要最準台灣口語 → **選 2**(首次用會下載約 3GB)

## 功能 1|語音輸入(即時打字)

1. 雙擊 `語音輸入.bat`,選模型,等「就緒」(黑視窗別關)
2. 游標點到要打字的地方(Word / Line / 瀏覽器…)
3. **按住 Alt+Ctrl 講話 → 放開** → 文字自動貼上
4. 離開:黑視窗按 `Esc`

**內建強化**(調整改 `voice_input.py` 最上面):
- **自動去靜音**:沒講話的段落不送辨識,靜音會顯示「未偵測到語音,略過」→ 少亂字
- **兩種模式**:`MODE="hold"` 按住講(預設);`MODE="toggle"` 按一下開、再按一下停(長段口述免一直按)
- **聽打歷史**:每次口述自動存 `聽打紀錄.txt`(含時間),貼失敗也救得回
- **換熱鍵**:改 `HOLD_KEYS`,例 `["ctrl","space"]`
- **小聲被略過**:把 `VAD_THRESHOLD` 從 0.5 調低(如 0.3)

---

## 功能 2|錄會議(麥克風 + 電腦輸出)

錄 Teams / Meet / 電話 / 任何線上會議,把**你的聲音 + 對方的聲音**一起錄下轉逐字稿。
原理:WASAPI loopback 直接抓喇叭輸出,**戴耳機也抓得到對方**,不用「立體聲混音」。

1. 雙擊 `錄會議.bat`,選模型
2. 看到 `🔴 開始錄音` 就開會,**全程一直錄**
3. 結束時回黑視窗**按 Enter** 停止
4. 自動:混音 → 存 wav → 轉逐字稿
5. 檔案預設在本資料夾的 `會議錄音\`
   - `會議錄音_日期_時間.wav`(原始錄音)
   - `會議錄音_日期_時間.txt`(逐字稿)

> 想換存檔位置:設環境變數 `TYPETOOL_REC_DIR` 指到你要的資料夾即可。

注意:
- 實體會議建議外接會議麥克風,筆電內建麥收遠端講者會漏字
- 說話人分離準度看講者聲音差異,雜訊多可能標錯 → 以文字內容為準

---

## 功能 3|轉逐字稿(已有音檔)

把 wav / mp3 / m4a / mp4 **拖到** `轉逐字稿_拖曳音檔到此.bat` → 選模型 → 同資料夾出現同名 `.txt`。

- 手機錄的 .m4a、Teams 下載的 .mp4 都直接可用
- 1 小時音檔(有 GPU)約 3–5 分鐘轉完;CPU 會慢很多

---

## 功能 4|說話人分離

把音檔**拖到** `說話人分離_拖曳音檔到此.bat` → 選模型。
選 1(SenseVoice)輸出 `[00:12] 說話人1:…`,存 `檔名_說話人.txt`。
選 2(Breeze)因不支援分離,改輸出 `[00:12] 文字`(只有時間戳,不標人)。

---

## 自訂取代字典(所有功能共用)

專業術語常被聽錯(Creo / Ansys / 線切割 / 專案代號)?
編輯 `dict.txt`,每行「錯詞=正確詞」:
```
快樂=Creo
安色斯=Ansys
現切割=線切割
```
辨識後自動替換,改完存檔、重開工具即生效。**四個功能都會套用。**

## 常見問題

| 問題 | 處理 |
|---|---|
| 沒有 NVIDIA 顯卡 | 可用,`install.bat` 自動裝 CPU 版;建議以 SenseVoice 為主 |
| CUDA 裝好跑不動 | 更新 NVIDIA 驅動;或改 CPU 版重裝 |
| Breeze 貼出重複/亂字 | 已內建防重複 + VAD 去靜音;仍有問題把該 wav 留著回報 |
| 語音輸入小聲被略過 | 調低 `VAD_THRESHOLD`(0.5 → 0.3) |
| 錄會議只錄到單邊 | 確認 Windows 預設播放/錄音裝置 |
| 中文變亂碼 | 腳本已強制 UTF-8,若異常回報 |

| 檔案 | 用途 |
|---|---|
| `install.bat` | 一鍵安裝(自動偵測顯卡) |

| `轉逐字稿_拖曳音檔到此.bat` / `transcribe_file.py` | 功能 3 |

| `requirements.txt` | 套件清單(install.bat 使用) |

## 關鍵版本(勿亂升級)

- Python 3.11、PyTorch 2.9.1、torchaudio 2.9.1
- funasr 1.3.14、**transformers 4.46.3(不可升 5.x)**、opencc、soundcard、silero-vad
- 不掛 ct-punc(SenseVoice 自帶標點);Breeze 讀檔走 ffmpeg 解碼
- 模型切換 / 字典 / 裝置偵測集中在 `asr_engine.py`
